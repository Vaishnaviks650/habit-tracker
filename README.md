# 🔥 Habit Tracker

A simple and modern Habit Tracker built using **HTML, CSS, and JavaScript**.

## Features

* ✅ Add new habits
* ✅ Mark habits as completed
* ✅ Delete habits
* ✅ Save habits using Local Storage
* ✅ Progress tracking
* ✅ Clean and responsive design

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.

No installation required.
